# ProjectCodeZ
Python 3.4

## Dice Simulator

This is a simple command line application that allows users to input values for a roll of dice. User inputs both number of dice (1-10) and number of sides (2-20). The results are then printed out.

#####Input example:
```
3d6
```

#####Output example:
```
Roll 1 : 4
Roll 2 : 5
Roll 3 : 4
Sum:  13
```
